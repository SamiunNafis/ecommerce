## Installation

**Basic React installation guide:**

Cloning the repo:
```shell
git clone https://github.com/SamiunNafis/ecommerce/
```
Head to the project directory
```shell
cd path/to/your/project/ecommerce
```
Installing _node_modules_ directory
```shell
npm i
```
Start the project
```shell
npm start
```
